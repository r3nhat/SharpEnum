# SharpEnum
C# Basic Enumeration

SharpEnum currently support the below commands:

* nslookup www.google.com
* ipconfig /all
* whoami /priv
* netstat -ano
* route PRINT
* net users
* net localgroup administrators
* dir c:\
* arp -a
* hostname
* echo %logonserver%
* schtasks /query /fo LIST 2>nul | findstr TaskName
* quser
